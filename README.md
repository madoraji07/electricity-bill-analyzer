# ⚡ 전기요금 분석 시스템

## 📝 프로젝트 소개
가정 및 사업장의 전기 사용량 데이터를 분석하여 요금을 계산하고, 사용 패턴을 **Streamlit 웹 대시보드**로 시각화하여 실시간 절약 방안을 제시하는 시스템입니다.

---

* **👨‍💻 팀원 1 - 이초한 (기여도: 33%)**
  - **담당:** 데이터 로딩 및 전처리 모듈 (`data_loader.py`)
  - **주요 기능:** 시뮬레이션용 샘플 데이터 생성, CSV 데이터 로딩 및 인코딩 예외 처리, 월별/요일별 데이터 집계 로직 구현
  - **커밋 수:** 9회

  - **커밋 수:** 회
* **👨‍💻 팀원 3 -  (기여도: 33%)**

  - **커밋 수:** 회

## 📊 주요 기능 상세 설명

### 1. 데이터 로딩 및 전처리
* **샘플 데이터 생성 (`generate_sample_data`)**
  - 실제 전기 사용 패턴을 정교하게 반영한 **1년 단위 시뮬레이션 데이터(CSV)**를 자동 생성합니다.
  - **여름철(7월~8월, 1.5배 가중치)** 및 **겨울철(12월~2월, 1.3배 가중치)** 냉난방 특성을 반영한 계절별 가중치를 적용했습니다.
  - 가정용(주말 1.2배 증가)과 사업장용(주말 0.4배 감소)의 사용자 타입별 패턴을 차별화했습니다.
  - 난수 고정(seed=42)을 통해 매번 일관된 테스트 데이터가 생성되도록 설계했습니다.
* **데이터 로딩 및 유효성 검사 (`load_data`)**
  - 외부 CSV 파일을 읽어올 때 발생할 수 있는 오류를 원천 차단하는 방어적 코드입니다.
  - `utf-8-sig`(BOM 제거)를 우선 적용하되, 실패 시 `cp949`로 교차 검증하여 **한글 깨짐 및 런타임 에러를 방지**합니다.
  - 필수 컬럼(`Date`, `Type`, `Usage_kWh`) 존재 여부를 검증하고, 날짜순 정렬 및 인덱스 재정렬을 자동화했습니다.
* **데이터 집계 및 정제 (`aggregate_monthly`, `aggregate_weekly`)**
  - 분석 및 시각화 담당 팀원들이 데이터를 즉시 활용할 수 있도록 규격화된 요약본을 정제합니다.
  - 연도별, 월별, 타입별로 그룹화하여 **[총사용량, 일평균사용량, 최대사용량]**을 산출하고 소수점 둘째 자리로 포맷팅했습니다.
  - 기획서 요구사항에 따라 **요일별 일평균 사용량**을 집계하고, 월요일부터 일요일까지 순서가 강제 정렬되도록 구현했습니다.

### 2. 요금 계산 및 실시간 분석
* **한국전력 주택용 고압 누진제** 기반의 정밀 요금 산정 알고리즘 적용
* 사이드바 **실시간 전력량 입력창**을 통해 사용자가 숫자를 변경하면 기본요금, 전력량요금, 부가세(10%), 전력기금(3.7%)을 **즉각 연산 및 동적 업데이트**
* 전체 시계열 데이터를 추적하여 연간 통계 및 월별 청구 금액 누적 데이터프레임 자동 생성

### 3. 웹 대시보드 시각화
* Streamlit 레이아웃(Sidebar, Tabs, Expander)을 적용하여 직관적이고 트렌디한 UI/UX 구현
* 연간 월별 사용량 추이 및 요금 비교 막대 그래프 제공
* 평일과 주말의 소비 성향을 한눈에 파악하는 **요일별 패턴 분석 차트** (한글 폰트 깨짐 방지 내장)
* 현재 설정된 사용량 기준 요금 항목별 지출 비중을 보여주는 **파이 차트(Pie Chart)**

### 4. 맞춤형 절약 가이드
* 사용자의 입력량에 따른 **현재 누진제 구간(1, 2, 3구간) 실시간 분석 및 시각적 경고** 제시
* 하위 누진 구간으로 진입할 수 있는 목표 사용량을 가이드하고, 진입 성공 시 **절감 가능한 예상 금액을 화폐 단위로 실시간 산출**
