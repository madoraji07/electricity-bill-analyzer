# ⚡ 전기요금 분석 시스템 - 데이터 로딩 및 전처리 모듈

* **작업 브랜치:** feature/data-loader
* **담당자:** 이초한 (기여도 33%, 커밋 수 4회)

본 브랜치는 전체 시스템의 기반이 되는 데이터의 생성, 로딩, 정제 과정(src/data_loader.py)을 전담하여 구현한 공간입니다. AI와의 **바이브코딩**을 통해 도메인 지식을 반영한 시뮬레이션 데이터를 설계하고, **방어적 코딩**을 통해 데이터의 안정성을 확보했습니다.

---

## 📊 구현 기능 상세 설명

교수님 지시사항 및 역할 분담에 따라 아래 **3가지 핵심 기능**을 개발 완료했습니다.

### 1. 샘플 데이터 생성 (generate_sample_data)
* 실제 전기 사용 패턴을 정교하게 반영한 1년 단위 시뮬레이션 데이터(CSV)를 자동 생성합니다.
* **여름(7월에서 8월, 1.5배)** 및 **겨울(12월에서 2월, 1.3배)** 냉난방 특성을 반영한 **계절별 가중치**를 적용했습니다.
* 가정용(주말 1.2배 증가)과 사업장용(주말 0.4배 감소)의 사용자 타입별 패턴을 차별화했습니다.
* 난수 고정(seed=42)을 통해 매번 **일관된 테스트 데이터**가 생성되도록 설계했습니다.

### 2. 데이터 로딩 및 유효성 검사 (load_data)
* 외부 CSV 파일을 읽어올 때 발생할 수 있는 오류를 원천 차단하는 **방어적 코드**입니다.
* utf-8-sig(BOM 제거)를 우선 적용하되, 실패 시 **cp949로 교차 검증**하여 **한글 깨짐 및 런타임 에러를 방지**합니다.
* **필수 컬럼(Date, Type, Usage_kWh) 존재 여부를 검증**하고, 날짜순 정렬 및 인덱스 재정렬을 자동화했습니다.

### 3. 월별 데이터 집계 (aggregate_monthly)
* 분석 및 시각화 담당 팀원들이 데이터를 즉시 활용할 수 있도록 규격화된 요약본을 정제합니다.
* 연도별, 월별, 타입별로 그룹화하여 [총사용량, 일평균사용량, 최대사용량]을 산출합니다.
* **소수점 둘째 자리로 포맷팅**하여 데이터의 가독성과 전달력을 높였습니다.

## 🚀 모듈 단독 실행 및 테스트 방법

`data_loader.py` 파일 하단에 **자체 검증용 테스트 코드**가 포함되어 있어, 메인 프로그램 통합 전에도 **단독으로 구동 확인이 가능**합니다.

**1. 필수 라이브러리 설치** pip install pandas numpy

**2. 데이터 로더 모듈 단독 실행** python src/data_loader.py

* 실행 시 임시 파일(`electricity_usage_test.csv`)을 생성하여 **[데이터 생성 -> 로딩 -> 월별 집계]** 과정을 콘솔에 출력한 후, **임시 파일을 안전하게 자동 삭제**합니다.
python src/data_loader.py
