# ⚡ 전기요금 분석 시스템

## 프로젝트 소개
가정/사업장의 전기 사용량 데이터를 분석하여 요금을 계산하고, 
사용 패턴을 시각화하여 절약 방안을 제시하는 시스템입니다.

## 주요 기능

### 1. 📊 데이터 분석
- 일별/월별 전기 사용량 집계
- 요일별 사용 패턴 분석
- 계절별 변화 추이

### 2. 💰 요금 계산
- 한국전력 누진제 기반 요금 산정
- 구간별 상세 내역 제공
- 부가세 및 전력기금 자동 계산

### 3. 📈 시각화
- 일별 사용량 라인 차트
- 월별 비교 막대 그래프
- 요일별 패턴 분석
- 요금 구성 파이 차트

### 4. 💡 절약 팁
- 누진 구간별 맞춤 조언
- 절감 가능 금액 계산
- 실용적인 절약 방법 제시
