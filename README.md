# ⚡ 전기요금 분석 시스템

## 📝 프로젝트 소개
가정 및 사업장의 전기 사용량 데이터를 분석하여 요금을 계산하고, 사용 패턴을 시각화하여 절약 방안을 제시하는 시스템입니다.

## 📊 주요 기능 상세 설명

### 1. 데이터 로딩 및 전처리
* **샘플 데이터 생성 (`generate_sample_data`)**
  - 실제 전기 사용 패턴을 정교하게 반영한 **1년 단위 시뮬레이션 데이터(CSV)**를 자동 생성합니다.
  - **여름철(7월에서 8월, 1.5배 가중치)** 및 **겨울철(12월에서 2월, 1.3배 가중치)** 냉난방 특성을 반영한 계절별 가중치를 적용했습니다.
  - **가정용(주말 1.2배 증가)**과 **사업장용(주말 0.4배 감소)**의 사용자 타입별 패턴을 차별화했습니다.
  - **난수 고정(seed=42)**을 통해 매번 일관된 테스트 데이터가 생성되도록 설계했습니다.
* **데이터 로딩 및 유효성 검사 (`load_data`)**
  - 외부 CSV 파일을 읽어올 때 발생할 수 있는 오류를 원천 차단하는 방어적 코드입니다.
  - `utf-8-sig`(BOM 제거)를 우선 적용하되, 실패 시 `cp949`로 교차 검증하여 한글 깨짐 및 런타임 에러를 방지합니다.
  - 필수 컬럼(`Date`, `Type`, `Usage_kWh`) 존재 여부를 검증하고, 날짜순 정렬 및 인덱스 재정렬을 자동화했습니다.
* **월별 데이터 집계 (`aggregate_monthly`)**
  - 분석 및 시각화 담당 팀원들이 데이터를 즉시 활용할 수 있도록 규격화된 요약본을 정제합니다.
  - 연도별, 월별, 타입별로 그룹화하여 **[총사용량, 일평균사용량, 최대사용량]**을 산출하고 소수점 둘째 자리로 포맷팅했습니다.

### 2. 요금 계산 
* 한국전력 누진제 기반 요금 산정
* 구간별 상세 내역 제공
* 부가세 및 전력기금 자동 계산

### 3. 시각화 
* 일별 사용량 라인 차트 및 월별 비교 막대 그래프
* 요일별 패턴 분석 및 요금 구성 파이 차트

### 4. 절약 팁
* 누진 구간별 맞춤 조언 및 실용적인 절약 방법 제시
* 절감 가능 금액 계산
