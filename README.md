# ⚡ 전기요금 분석 및 인터랙티브 절약 대시보드 시스템

## 📝 프로젝트 소개
가정 및 사업장의 전기 사용량 데이터를 분석하여 요금을 계산하고, 사용 패턴을 Streamlit 웹 대시보드로 시각화하여 실시간 절약 방안을 제시하는 시스템입니다. 파이썬 백엔드 모듈과 인터랙티브 UI가 유기적으로 결합되어 동작합니다.

## 👥 팀원 및 역할 분담

* 👨‍💻 **팀원 1 - 이초한 (기여도: 33%)**
  * **담당:** 데이터 로딩 및 전처리 모듈 (`data_loader.py`)
  * **주요 기능:** 시뮬레이션용 샘플 데이터 생성, CSV 데이터 로딩 및 인코딩 예외 처리, 월별/요일별 데이터 집계 로직 구현
  * **커밋 수:** 9회

* 👩‍💻 **팀원 2 - 마연희 (기여도: 33%)**
  * **담당:** 요금 계산 및 분석 로직 (`analyzer.py`)
  * **주요 기능:** 주택용 고압 누진제 요금 계산 알고리즘, 실시간 통계 분석 기능, 맞춤형 절약 팁 알고리즘 설계
  * **커밋 수:** 7회

* 👩‍💻 **팀원 3 - 조민서 (기여도: 33%)**
  * **담당:** 시각화 및 메인 프로그램 (`visualizer.py`, `main.py`)
  * **주요 기능:** Streamlit 인터랙티브 UI 구축, 데이터 연동형 그래프 생성 (matplotlib/seaborn), 실시간 입력 제어 인터페이스 및 통합 대시보드 구현
  * **커밋 수:** 8회

## 📊 주요 기능 상세 설명

### 1. 데이터 로딩 및 전처리 (`data_loader.py`)
* **샘플 데이터 생성 (`generate_sample_data`):** 실제 전기 사용 패턴을 정교하게 반영한 1년 단위 시뮬레이션 데이터(CSV)를 자동 생성합니다. 여름철(7월부터 8월 사이, 1.5배) 및 겨울철(12월부터 익년 2월 사이, 1.3배) 냉난방 특성을 반영하고 가정을 비롯한 사용자 타입별 패턴을 차별화했습니다. 난수 고정(seed=42)을 통해 일관된 테스트 데이터를 생성합니다.
* **데이터 로딩 및 유효성 검사 (`load_data`):** 외부 CSV 파일을 읽어올 때 발생할 수 있는 인코딩 오류를 방지하기 위해 `utf-8-sig`를 우선 적용하되, 실패 시 `cp949`로 교차 검증합니다. 필수 컬럼 검증 및 날짜순 정렬을 자동화했습니다.
* **데이터 집계 및 정제 (`aggregate_monthly`, `aggregate_weekly`):** 연도별, 월별, 타입별로 그룹화하여 `[총사용량, 일평균사용량, 최대사용량]`을 산출하고 요일별 일평균 사용량이 월요일부터 일요일 순서로 강제 정렬되도록 정제합니다.

### 2. 요금 계산 및 실시간 분석 (`analyzer.py`)
* 한국전력 주택용 고압 누진제 기반의 정밀 요금 산정 알고리즘을 적용했습니다.
* 사이드바 실시간 전력량 입력창을 통해 사용자가 숫자를 변경하면 기본요금, 전력량요금, 부가세(10%), 전력기금(3.7%)을 즉각 연산하여 동적으로 실시간 요금을 업데이트합니다.
* 전체 시계열 데이터를 추적하여 연간 통계 및 월별 청구 금액 누적 데이터프레임을 자동 생성합니다.

### 3. 웹 대시보드 시각화 및 가이드 (`visualizer.py`, `main.py`)
* Streamlit 레이아웃을 적용하여 직관적이고 트렌디한 UI/UX를 구현했습니다.
* 연간 월별 사용량 추이 막대 그래프 및 평일과 주말의 소비 성향을 파악하는 요일별 패턴 분석 차트를 제공합니다.
* 사용자의 입력량에 따른 현재 누진제 구간(1, 2, 3구간) 실시간 분석 및 시각적 경고를 제시하며 하위 누진 구간으로 진입할 수 있는 목표 사용량과 절감 가능 금액을 실시간 산출합니다.
